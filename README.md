Eye tracking C++ library

### Directory structure
- `bin ............` binaries
- `cb .............` Code::Blocks projects
- `doc ............` documentation
- `include ........` library interface headers
- `lib ............` static library
- `src ............` source code
- `test ...........` unit tests
